The Rescue Payload

An AI-powered proactive task rescue assistant that transforms chaotic thoughts, messages, and deadlines into structured, prioritized action plans using Google Gemini.

📌 Overview

The Rescue Payload is an intelligent productivity assistant designed to help users rescue important tasks before they become overdue.

Instead of manually organizing notes, emails, Slack messages, or scattered thoughts, users simply paste unstructured text into the application. Gemini AI extracts actionable tasks, estimates effort, identifies deadlines, assigns priorities, and continuously monitors progress through an interactive dashboard.

The application acts as a personal AI operations center that helps users stay ahead of deadlines and reduce cognitive overload.


Features


🤖 AI-powered task extraction from unstructured text

📅 Automatic deadline identification

🚨 Smart priority assignment (Critical, High, Medium, Low)

⏱️ Time estimation for every task

📊 Live productivity dashboard

📈 Progress and pending task monitoring

🔄 Task status management

💡 AI-generated structured task summaries

🌙 Modern cyber-inspired responsive UI

☁️ Cloud deployment using Google Cloud Run


🚀 How It Works


User pastes raw thoughts, emails, notes, or messages.

Gemini AI analyzes the text.

Important tasks are extracted.

Deadlines and estimated effort are identified.

Tasks are prioritized automatically.

The dashboard displays pending work and remaining time.

Users update task status until completion.
